# Edo Travel - Сайт турагентства

Современный 5-страничный сайт турагентства на Next.js с Tailwind CSS.

## Страницы

- **Главная** (`/`) - Главная страница с hero-секцией, преимуществами и популярными направлениями
- **О нас** (`/about`) - Информация о компании, миссия и достижения
- **Туры** (`/tours`) - Каталог туров с описанием и ценами
- **Галерея** (`/gallery`) - Галерея направлений и отзывы клиентов
- **Контакты** (`/contacts`) - Контактная информация и форма обратной связи

## Технологии

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React 18

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен версии
npm start
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Деплой на Vercel

Проект готов к деплою на Vercel:

1. Загрузите проект в GitHub
2. Подключите репозиторий к Vercel
3. Vercel автоматически определит Next.js и настроит деплой

Или используйте Vercel CLI:

```bash
npm i -g vercel
vercel
```
